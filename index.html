HTML

HTML, or Hypertext Markup Language, is the standard markup language for creating web pages. HTML is used to structure the content of a web page, such as paragraphs, lists, images, and data tables.

CSS

CSS stands for Cascading Style Sheets, and it's a computer language for structuring and laying out web pages. CSS uses coding elements and "cascading style sheets" to describe how documents written in HTML or XML should be presented.

Javascript

JavaScript is the Programming Language for the Web. JavaScript can update and change both HTML and CSS. JavaScript can calculate, manipulate and validate data.
